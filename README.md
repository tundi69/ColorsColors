# Code-Breaking Game (JavaFX)

A desktop version of the classic code-breaking board game, where the player must guess a hidden sequence of four colors generated by the computer.

After each guess, the player receives feedback indicating:
- how many colors are correct but in the wrong position
- how many colors are correct and in the correct position

Colors may be repeated, which adds an extra layer of challenge and requires logical deduction to solve the sequence.

## Features
- Interactive JavaFX-based graphical user interface
- Feedback system to guide the player toward the correct solution
- Simple but engaging gameplay based on logical reasoning

## Technologies
- Java
- JavaFX
